# Test di Connessione TCP e UDP

Questo progetto fornisce una semplice interfaccia grafica (GUI) per testare la connessione verso un determinato indirizzo IP e porta utilizzando i protocolli **TCP** o **UDP**.

## Struttura del Progetto
- **main.py**: Contiene la funzione di avvio dell’applicazione (`start_app()`).
- **gui.py**: Gestisce l’interfaccia grafica con Tkinter.
- **connection_tester.py**: Implementa le funzioni per testare le connessioni TCP e UDP.
- **requirements.txt (opzionale)**: Elenca eventuali librerie necessarie (se servirà in futuro aggiungere librerie non di default).

## Requisiti
- Python 3.x
- Modulo `tkinter` (incluso di default in molte installazioni di Python, ma su alcuni sistemi potrebbe dover essere installato separatamente).
